# Car Price Prediction Using Machine Learning

## Project Overview
This project predicts the price of a car using Machine Learning algorithms based on various car features.

## Files Included
- car_data.csv : Dataset used for training and testing.
- car_price_prediction.py : Python script for data preprocessing, training, and prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps to Run
1. Clone the repository:
   git clone https://github.com/asin758/Car-Price-Prediction-With-Machine-Learning.git

2. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib

3. Run the program:
   python car_price_prediction.py

## Machine Learning Model
The project uses regression techniques to predict car prices based on the provided dataset.
